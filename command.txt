The command I used to compile this project:
Dev debug: gcc -g -std=c18 -Wall WiiUDownloader.c -lcurl -lws2_32 -o WiiUDownloader -D__USE_MINGW_ANSI_STDIO
Release: gcc -s -Os -std=c18 -Wall WiiUDownloader.c libcurl.a -lws2_32 -o WiiUDownloader -DCURL_STATICLIB -D__USE_MINGW_ANSI_STDIO

Note that this will likely only work in an environment similar to mine (at least the release one), as the libcurl.a was compiled by myself specifically with MinGW64.
To compile a similar libcurl.a, download the source from https://curl.haxx.se/download.html . See https://curl.haxx.se/docs/install.html for options.
I used "./configure --disable-ftp --disable-ldaps --disable-ldsp --disable-file --disable-dict --disable-proxy --disable-unix-sockets --disable-verbose --disable-versioned-symbols --disable-telnet --disable-tftp --disable-rtsp --disable-pop3 --disable-imap --disable-gopher --disable-smtp --disable-smb --disable-ipv6 --disable-crypto-auth --disable-cookies --disable-tls-srp --disable-manual --with-gnu-ld --without-brotli --without-zlib --without-winssl --without-schannel --without-ssl --without-mbedtls --without-libpsl --without-winidn --without-libidn2 --without-librtmp --without-quiche --without-ca-bundle --without-ca-path --disable-ares --disable-ldap --without-wolfssl --without-cyassl --without-mesalink --disable-shared --disable-mime --disable-sspi --disable-nss --disable-pthreads" as configuration options (followed by "make install").
I compiled version 7.66.0, which is the most recent version as of the time of this writing.

What I used for linux libcurl compilation: "./configure --disable-ftp --disable-ldaps --disable-ldsp --disable-file --disable-dict --disable-proxy --disable-unix-sockets --disable-verbose --disable-versioned-symbols --disable-telnet --disable-tftp --disable-rtsp --disable-pop3 --disable-imap --disable-gopher --disable-smtp --disable-smb --disable-ipv6 --disable-crypto-auth --disable-cookies --disable-tls-srp --disable-manual --with-gnu-ld --without-brotli --without-zlib --without-winssl --without-schannel --without-ssl --without-mbedtls --without-libpsl --without-winidn --without-libidn2 --without-librtmp --without-quiche --without-ca-bundle --without-ca-path --disable-ares --disable-ldap --without-wolfssl --without-cyassl --without-mesalink --disable-shared --disable-mime --disable-sspi --disable-nss --disable-pthreads --disable-threaded-resolver"
